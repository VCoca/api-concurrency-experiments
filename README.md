# API Concurrency Experiments

Collection of backend projects exploring different approaches to API communication and concurrency in C#.

## Projects

### GitHub Commit Statistics Server — Task-based
Retrieves repository commit statistics using the GitHub API with task-based asynchronous programming.

### GitHub Commit Statistics Server — ThreadPool-based
Alternative implementation using the .NET ThreadPool for concurrent request handling.

### SportMonks Reactive Client
Project exploring reactive programming concepts while working with the SportMonks API.

## Technologies

- C#
- .NET
- REST APIs
- Task-based concurrency
- ThreadPool
- Reactive programming

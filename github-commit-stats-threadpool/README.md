# GitHub Commit Statistics Server

A simple web server that retrieves commit statistics for a GitHub repository using the GitHub API.

## Features

- Fetches repository commit data from the GitHub API
- Processes JSON responses
- Returns structured commit statistics
- Uses a ThreadPool to handle multiple requests

## Technologies

- C#
- REST API
- JSON processing

## How it works

The server sends HTTP requests to the GitHub API to retrieve commit data for a repository.
The response is parsed and processed to extract useful statistics.
